# Arcanoid (Canvas)

Минимально играбельный арканоид на чистом Canvas 2D (JavaScript), с уровнями и простыми SFX через WebAudio.

## Ссылка на игру

`https://easy-arcanoid.netlify.app/`

## Запуск

Нужен локальный сервер.

- **VS Code / Cursor**: Live Server → открыть `index.html`
- **Node** (если установлен):

```bash
npx serve .
```

Затем открыть адрес, который покажет сервер.

## Управление

- **Мышь / тач**: движение платформы по X (следует за указателем)
- **Клавиатура**: `←` / `→` двигают платформу
- **Запуск шара**: клик/тап или `Space`
